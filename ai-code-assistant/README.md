# AI Code Assistant

Небольшой MVP-проект на **Python + FastAPI**, который принимает вопрос или код и отправляет запрос в LLM API.
Подходит как pet-project под вакансии, связанные с backend, AI-интеграциями и LLM.

## Что умеет
- объяснять код
- искать ошибки
- предлагать улучшения
- генерировать код по текстовому запросу

## Что значит "интеграции"
В этом проекте интеграция — это подключение **внешнего сервиса** через API.

Здесь есть интеграция с:
- OpenAI-compatible API
- любым совместимым LLM endpoint, который принимает chat/completions-запросы

То есть твой backend:
1. принимает запрос от пользователя
2. формирует prompt
3. отправляет его во внешний AI API
4. возвращает ответ клиенту

Это и есть базовая AI-интеграция.

## Структура
- `app/main.py` — точка входа FastAPI
- `app/routers/assistant.py` — API роуты
- `app/services/llm_client.py` — клиент для работы с LLM API
- `app/core/config.py` — настройки
- `app/schemas.py` — Pydantic-схемы

## Запуск локально

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 2. Создай `.env`
```bash
cp .env.example .env
```

### 3. Запуск
```bash
uvicorn app.main:app --reload
```

Открыть:
- API: `http://127.0.0.1:8000`
- Swagger: `http://127.0.0.1:8000/docs`

## Пример запроса
POST `/assistant/ask`

```json
{
  "mode": "explain",
  "prompt": "Объясни этот код",
  "code": "def add(a, b):\n    return a + b"
}
```

## Режимы
- `explain` — объяснение кода
- `fix` — поиск ошибок и предложение исправления
- `generate` — генерация кода по описанию
- `review` — код-ревью
