#  AI Code Assistant

MVP AI-ассистента для разработчиков на FastAPI. Помогает объяснять код, исправлять ошибки, генерировать решения и проводить код-ревью.

##  О проекте

AI Code Assistant — это REST API сервис, который использует LLM для помощи разработчикам в повседневных задачах:

- **Объяснение кода** — разбор сложных фрагментов
- **Исправление ошибок** — поиск и исправление багов
- **Генерация кода** — создание решений по описанию
- **Код-ревью** — анализ качества и рекомендации

##  Режимы работы

| Режим | Описание | Пример запроса |
|-------|----------|----------------|
| `explain` | Объяснение кода | "Что делает эта функция?" |
| `fix` | Исправление ошибок | "Найди баг в этом коде" |
| `generate` | Генерация кода | "Напиши функцию сортировки" |
| `review` | Код-ревью | "Оцени качество кода" |

## Технологии

- **Framework**: FastAPI 0.104+
- **Валидация**: Pydantic v2
- **Асинхронность**: async/await
- **Документация**: OpenAPI (Swagger UI)

##  Установка и запуск

### 1. Клонирование

git clone https://github.com/ВАШ_ЛОГИН/ai-code-assistant.git
cd ai-code-assistant

Установка зависимостей
pip install -r requirements.txt

Настройка переменных окружения
APP_NAME="AI Code Assistant"
OPENAI_API_KEY=your-api-key-here
MODEL_NAME=gpt-3.5-turbo

Запуск сервера
uvicorn main:app --reload --port 8000
